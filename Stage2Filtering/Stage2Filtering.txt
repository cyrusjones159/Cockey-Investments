🐍 Python: 10 Years of Prices & Dividends (Yahoo Finance)
python
import yfinance as yf

ticker = yf.Ticker("AAPL")

# 10 years of daily prices
prices = ticker.history(period="10y")

# Dividend history
dividends = ticker.dividends

print(prices.head())
print(dividends.head())
Pros:

No API key

Reliable for major tickers

Includes dividends, splits, corporate actions

📊 R: 10 Years of Prices & Dividends (quantmod)
r
library(quantmod)

symbol <- "AAPL"

# 10 years of price data
getSymbols(symbol, from = Sys.Date() - 3650, auto.assign = TRUE)
prices <- get(symbol)

# Dividend history
divs <- getDividends(symbol, from = Sys.Date() - 3650)

head(prices)
head(divs)
Pros:

Very easy

Built-in dividend retrieval

Works with many tickers

🔑 Python: Using Alpha Vantage (Free API Key)
python
import requests

API_KEY = "YOUR_KEY"
symbol = "AAPL"

# Prices (daily adjusted)
url_prices = f"https://www.alphavantage.co/query?function=TIME_SERIES_DAILY_ADJUSTED&symbol={symbol}&apikey={API_KEY}"
prices = requests.get(url_prices).json()

# Dividends are included in "dividend amount" field
